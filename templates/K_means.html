Unsupervised Learning - K-Means

K-Means is an unsupervised learning algorithm used to group data into clusters based on similarity.

How it works
  • Select number of clusters (k)
  • Initialize centroids
  • Assign points to nearest centroid
  • Update centroids
  • Repeat until convergence
Dataset Description

The dataset represents customers based on:

  • Age: Customer age
  • Credit Limit: Financial capacity
Interpretation
  • Cluster 0 → Low credit customers
  • Cluster 1 → Medium credit customers
  • Cluster 2 → High credit customers

The algorithm groups customers with similar financial profiles, helping identify different segments.
