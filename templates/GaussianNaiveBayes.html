Gaussian Naive Bayes

Probabilistic classification for continuous data

What is it?

Gaussian Naive Bayes is a variant of the Naive Bayes algorithm designed to handle continuous variables. It is based on Bayes' Theorem and assumes that the features for each class follow a normal distribution (or Gaussian distribution).

The Normality Assumption

When working with continuous data (such as height, weight, or temperature), we cannot simply count frequencies. Instead, we calculate the probability using the probability density function of the normal distribution:

P(x_i | y) = (1 / sqrt(2πσ²_y)) * exp(-(x_i - μ_y)² / 2σ²_y)
  • μ_y: The mean of the values in x for class y.
  • σ²_y: The variance of the values in x for class y.

Pros and Cons

Pros Cons
Extremely fast performance. Assumes independence between features (rare in real life).
Works well even with small datasets. Performance drops if data is not truly Gaussian.
Easy to implement and tune. Highly sensitive to outliers.

Common Use Cases

  • Text classification (when converted to continuous values like TF-IDF).
  • Initial medical diagnosis based on clinical metrics.
  • Spam detection and content filtering.
  • Basic recommendation systems.
