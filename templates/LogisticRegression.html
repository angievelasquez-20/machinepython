How Logistic Regression works

Unlike Linear Regression which predicts a continuous number, Logistic Regression predicts the probability of a categorical outcome (Yes/No, 0 or 1).

The Sigmoid Function

The model uses the Sigmoid function to map any real-valued number into a value between 0 and 1:

P(y=1) = \frac{1}{1 + e^{-(z)}}
