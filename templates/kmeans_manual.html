K-Means Manual Simulation

Introduction

In this activity, we implemented the K-Means clustering algorithm manually using Excel. The objective was to understand how clustering works through distance calculations, centroid updates, and variance reduction across iterations.

Dataset Description

The dataset consists of 100 records with two numerical features: Revenues and Profits. These variables represent financial performance and allow us to group similar entities.

Dataset in Excel

Iteration 1

Three initial centroids were selected manually. Then, the Euclidean distance between each data point and the centroids was calculated. Each point was assigned to the nearest cluster.

Distance calculations and cluster assignment

Iteration 2

New centroids were calculated as the average of each cluster. Using these updated centroids, distances were recalculated and clusters reassigned.

Updated clusters after recalculation

Iteration 3

The process was repeated once more. At this stage, the clusters became more stable, indicating convergence of the K-Means algorithm.

Final clustering result

Variance Analysis

The variance was calculated for each iteration. A decrease in variance indicates that the data points are getting closer to their respective centroids.

Variance reduction graph

Conclusion

The K-Means algorithm successfully grouped the dataset into clusters. Through iterative refinement, the model reduced variance and improved cluster compactness. This demonstrates how unsupervised learning can identify hidden patterns in data.
