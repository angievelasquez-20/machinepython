Machine Learning & Reinforcement Learning

Reinforcement Learning (RL)

Unlike other methods, RL does not require prior labeled data. The system learns through direct interaction with an environment.

Critical Components:

  • State: The current situation the agent perceives from the environment.
  • Action: What the agent decides to do based on its policy.
  • Reward: The scalar feedback it receives (positive or negative).
  • Policy: The strategy the agent follows to decide the next action.

The Bellman Equation (Simplified):

$V(s) = \max_a (R(s, a) + \gamma V(s'))$

Where the value of a state depends on the immediate reward plus the discounted future value ($\gamma$).

Context: Other Types

To understand RL, we must compare it with the other existing types of ML:

  • Supervised: The model has a teacher (labels).
  • Unsupervised: The model looks for structures on its own.
  • RL: The model learns from the consequences of its actions.

RL Applications

  • Recommendation Systems: Long-term click optimization.
  • Gaming: Defeating champions in Chess, Go, or Dota 2.
  • Trading: Dynamic financial portfolio management.
  • Control: Stabilization of drones and robotic arms.

Image Processing in RL

When tasked with processing an image, RL agents explore the entire image as a starting point and then sequentially identify objects until all aspects are registered. Computer vision systems also utilize deep convolutional neural networks, built from large labeled datasets, to map images to human-generated scene descriptions from simulation engines.

Key Examples:

  • Autonomous Vehicles: Environment mapping for navigation.
  • Medical Imaging: Identifying anomalies in diagnostic images.
  • Image Classification: Object detection and categorization.
